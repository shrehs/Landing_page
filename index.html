Image Classifiers

Landmark Classification & Tagging

Developed a CNN from scratch and utilized transfer learning with ResNet-18 for accurate landmark classification. This project involved data preprocessing, model deployment using TorchScript, and leveraging AWS SageMaker for training.

Bike Sharing Demand Prediction
